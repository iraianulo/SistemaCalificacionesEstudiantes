\# Gestor de Calificaciones en Java



Proyecto desarrollado en DAW.

Programa que gestiona notas usando arrays tridimensionales.



\## Qué hace:

Parte 1: Diseño de un programa para gestionar las calificaciones de estudiantes en diferentes asignaturas durante varios trimestres.

Parte 2: Diseño de un programa que solicite al usuario introducir una frase y realice operaciones con ella.



\## Tecnologías

\- Java

\- Arrays 3D

\- Math.random()



\## Qué he practicado

\- Estructuras multidimensionales

\- Cálculo de medias

\- Lógica de control
